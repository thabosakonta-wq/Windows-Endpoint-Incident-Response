# Executive Summary

## Incident Title

Suspicious PowerShell Activity on Windows Endpoint

## Severity

High

## Incident Overview

A security alert was triggered after suspicious PowerShell activity was detected on a Windows endpoint.

The investigation focused on determining whether the activity represented legitimate administration or malicious execution.

## Impact

The investigation identified suspicious command execution behaviour requiring further analysis.

## Response Actions

The SOC analyst performed:

- Alert validation
- Endpoint investigation
- Event log analysis
- Process review
- Detection mapping
- Incident documentation

## Outcome

The incident was contained and documented following the incident response lifecycle.
